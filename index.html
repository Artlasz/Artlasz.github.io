Full Body Base Model

This was my first time with a project of this scale and it proved very challenging, but through weeks of hard work I managed to get the grip of modelling more detailed characters like this.


This project followed the workflow of blockout, sculpt and retopology.

Making the blockout is simple enough, as it comprises of using a reference to mold simple shapes into the general shape of whatever you want to model.
After this comes sculpting - which I was already familiar with due to previous projects, but I had never used it in tandem with retopology before. During the sculpting phase you turn the blockout from a general shape into the desired shape, refining it and adding a great deal of detail. Sculpting is usually a lengthy procedure, but it gives great control to the modedeller to make fine details.
Lastly came the retopology - which was a massive challenge! Retopology is done to reduce the amount of vertexes on the model to allow for animation, but without sacrificing much detail. This has to be done due to sculpts having an incredibly high amount of vertexes, which is incompatible with animation. It consists of drawing vertexes over the sculpt for hours on end until the entire model is drawn over - after which you can then rig and weightpaint the model to make it possible to animate. Retopology also allows you to easily edit the model without having to use sculpting tools, this is because you keep the detail but have far fewer vertexes - which translates to more control over the model. The issue was that I had never done retopology on such a scale before and it took a very long time to accomplish with good quality in mind, but the end result was fantastic, with good edge flow and topology.
It was a very difficult project, but very worth it with how much I learned from it.

The biggest challenge to overcome was retopology and understanding good edge flow and topology - but due to the experience from this project, I managed to learn these concepts and put them into practice.

The tool used was Blender.

Hands

I modelled these hands for the Full Body Base Model seen earlier, hands have a bad reputation for being hard to make in art - but I found it to be enjoyable!


This project used a slightly altered workflow where we use a blockout, edit into the desired shape, use the subdivision modifier and then sculpt the shape and add the finer details.

Modelling the hands proved to be to somewhat difficult, but I found it to be more entertaining than annoying. Using the subdivision modifier helped greatly, as it allows you to use a blockier shape - which the modifier smoothens to allow for clean sculpting. The hands were much faster to model than the body itself, due to being a smaller part of the bigger picture.

The greatest challenge for this one was definitely the fingers, it took much sculpting and editing to make them look accurate. But the end result was very nice!

Christmas Wreath

I modelled this Christmas Wreath during the winter season for an assignment I was given.


For this one I used a lot of modifiers to speed up the process greatly - so instead of modelling each one of those dozens and dozens of branches, I modelled a few, positioned them as an outer ring and used modifiers to make them into a continuous circle and spread them out a little. The bells however only use the subdivision modifier, as well as having a nice metal shine. Lastly, there's all the lights on the wreath which I placed by hand where I thought they would look nice.

Another interesting part of this project is the background! It's not just a wall or a singular colour, rather it is a rolling background reminiscent of the ones professional photographers use in real life - to spread the light out and give some nice depth to the image. You can see this in greater detail if you go to my Cara page through the full portfolio button.

Getting all the shadows, lights and reflections to look good took a lot of effort - but the final render came out phenomenal. I came out of this project with a good understanding of renders in Blender.

Low Poly Man

I modelled this low poly character as part of my second year evaluated project. I wanted to do something simple, but also good looking - so I settled for a low poly character with cel shading.
Part of this project was also to UV map, rig and animate the model - which I managed to accomplish in the allotted time period.


I hadn't ever tried cel shading before so it was a new experience for me - but I managed to get a general understanding of its implementation relatively quickly. The model itself is detailed to follow the laws of anatomy with a hint of stylisation, so despite the low poly nature - it looks clean with the help of smooth shader.

The cel shading was accomplished through the shader editor using shader nodes that use a solidify modifier to display the effect.
Then the animation was done using a standard humanoid rig I made from scratch, which I then animated by hand with the help of inverse kinematics.
Lastly the UV mapping was accomplished using a texture preview I made to display the UV map while adjusting it, as well unwrapping the map into distinc areas contained by seams. This way I was able to easily adjust each area to display their texture correcly and smoothly,


Project Info:
I made this model in Blender in 5 days by myself.
The tool used was Blender.
